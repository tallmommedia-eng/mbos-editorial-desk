# MBOS Editorial Desk v0.3

Test build for the WordPress Dashboard widget.

## New in 0.3

- Renamed plugin to **MBOS Editorial Desk**.
- Tries multiple Site Kit GA4 report shapes.
- Adds visible diagnostics when Site Kit returns no page rows.
- Still caches results for 12 hours and supports manual Refresh.

## Test

Install, activate, open WordPress Dashboard, then click **Refresh** in the MBOS Editorial Desk widget.
